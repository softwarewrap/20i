# `scripts/email`

## Overview

The `scripts/email` directory contains command-line tools for automating
email administration using the 20i API.

## Current Commands

### `create-forward.php`

Creates email forwarding rules.

## Design Goals

-   Safe automation
-   Deterministic CLI behavior
-   Minimal dependencies
-   Reusable libraries

## Future Work

-   List forwards
-   Delete forwards
